# Site Map

The catalogue hosts a site map of its public records and provides an index of records in JSON and XML formats:

**JSON** · [Open](https://antcat.antarcticanz.govt.nz/geonetwork/srv/api/sitemap?format=jsonld)

```text
https://antcat.antarcticanz.govt.nz/geonetwork/srv/api/sitemap?format=jsonld
```

**XML** · [Open](https://antcat.antarcticanz.govt.nz/geonetwork/srv/api/sitemap?format=xml)

```text
https://antcat.antarcticanz.govt.nz/geonetwork/srv/api/sitemap?format=xml
```
